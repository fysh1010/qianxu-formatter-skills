## 代码块演示

### JavaScript 示例

```javascript
// 千序排版器核心代码
const formatter = {
  name: '千序排版器',
  version: '4.0.0',
  
  format(markdown) {
    return this.parse(markdown);
  },
  
  parse(text) {
    // 解析 Markdown
    return this.render(text);
  },
  
  render(html) {
    return `<section>${html}</section>`;
  }
};

console.log(formatter.format('# Hello'));
```

### Python 示例

```python
# Python 示例代码
def hello_qianxu():
    """问候函数"""
    print("你好，千序！")
    
class Formatter:
    def __init__(self, name):
        self.name = name
        
    def format(self, text):
        return f"排版后: {text}"

# 使用示例
formatter = Formatter("千序排版器")
result = formatter.format("测试文本")
```

## 总结

千序排版器 v4.0 核心改进：

- **统一字体**：正文 14px，标题 16px
- **标题编号**：32px 橙色数字
- **关键词高亮**：橙色加粗
- **代码块**：绿色标签 + 黑色代码区
- **表格优化**：清晰边框 + 交替行色
- **手机友好**：行高 1.75 舒适阅读

希望你喜欢这个排版风格！
